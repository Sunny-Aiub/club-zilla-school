# Club Zilla Digital Platform

## Overview
Club Zilla Digital Platform is a planned digital platform for Club Zilla School. The project is currently in the discovery and requirements definition phase.

## Project details
- **Project name:** Club Zilla Digital Platform
- **Client:** Club Zilla School
- **Development company:** Appartisans Lab
- **Current phase:** Discovery and requirements definition

## Documentation foundation
The repository currently contains foundational documentation only. Production development must not begin until the MVP, Business Requirements Document (BRD), Software Requirements Specification (SRS), and architecture are approved.

- `docs/00-project-bible.md` — central project reference and terminology source.
- `docs/01-product-vision.md` — product vision, goals, users, and success criteria.
- `docs/04-user-roles.md` — initial user role definitions and access assumptions.
- `docs/06-business-rules.md` — proposed and unresolved business rules.
